Scorecast

github

Some friends and I created a mobile app called Scorecast, which brings the power of live scoring to amateur sports teams. We created a cross platform app using Expo and chose Firebase for the database. The app uses a templating engine which allows any kind of game to be scored: complex games like baseball or a simple card games like hearts.
