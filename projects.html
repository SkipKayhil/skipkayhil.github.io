My take on Tic Tac Toe. The game is written in functional javascript with css styling that implements material design. The game currently features a togglable Computer Player that uses a Min Max algorithm to find the best move. In the future I'd like to add multiple difficulty settings

Twitch IRC

Back when I dreamed of being a Twitch streamer, I wanted to write a bot that could moderate chat for me. I used Java and the Twitch API, first only running messages through the console, and later added a full GUI using JavaFX. After that I started working on expanding the app to work with any irc server and not just Twitch.
